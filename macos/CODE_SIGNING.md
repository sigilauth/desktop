## Code Signing (CI B12)

### Local Development

```bash
# Build without signing
swift build --configuration debug

# Run tests without signing
swift test
```

### CI Build (B12 GitHub Actions)

```bash
# Sign with Developer ID Application certificate
codesign --sign "Developer ID Application: Sigil Auth (TEAM_ID)" \
    --force \
    --options runtime \
    --entitlements SigilAuth.entitlements \
    --deep \
    --timestamp \
    SigilAuth.app

# Verify signature
codesign --verify --deep --strict --verbose=2 SigilAuth.app
spctl --assess --type execute --verbose=4 SigilAuth.app
```

## Notarization (CI B12)

### Submit for Notarization

```bash
# Create distributable ZIP
ditto -c -k --keepParent SigilAuth.app SigilAuth.zip

# Submit to Apple
xcrun notarytool submit SigilAuth.zip \
    --apple-id "APPLE_ID" \
    --team-id "TEAM_ID" \
    --password "@keychain:NOTARY_PASSWORD" \
    --wait

# Check status
xcrun notarytool info SUBMISSION_ID \
    --apple-id "APPLE_ID" \
    --team-id "TEAM_ID" \
    --password "@keychain:NOTARY_PASSWORD"
```

### Staple Notarization Ticket

```bash
# Attach notarization ticket to app
xcrun stapler staple SigilAuth.app

# Verify stapling
xcrun stapler validate SigilAuth.app
```
